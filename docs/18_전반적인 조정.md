1. 각 섹션당 최대 mt인 5를 커스텀 css로 더 넓힌다
```css
/* margin 마진 */
.mt-6 {
    margin-top: 4.5rem !important;
}

.mt-7 {
    margin-top: 6rem !important;
}

.mt-section {
    margin-top: 7.5rem !important;
}
```
